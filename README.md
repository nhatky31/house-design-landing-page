# 🏠 House Design Landing Page

Dự án thực hành cá nhân nhằm rèn luyện kỹ năng xây dựng giao diện web từ bản thiết kế (mockup) có sẵn. Dự án mô phỏng một trang web giới thiệu dịch vụ bất động sản (Housing).

## 🚀 Công nghệ sử dụng
*   **HTML5:** Sử dụng cấu trúc thẻ Semantic (`<header>`, `<section>`, `<footer>`,...).
*   **CSS3:** Áp dụng Flexbox và CSS Grid layout để chia bố cục trang.
*   **Thư viện ngoài:** FontAwesome (Icons), Google Fonts (Montserrat & Roboto).
*   **Javascript:** Owl Carousel 2 (Tạo hiệu ứng trượt ảnh), jQuery CountUp (Hiệu ứng nhảy số thống kê).

## 💡 Điểm nổi bật của dự án
*   Tổ chức cấu trúc file và thư mục chuẩn chỉ (tách biệt HTML, CSS, JS, Image).
*   Sử dụng thành thạo **CSS Grid** để xử lý các phần bố cục phức tạp (Blog, Team, Contact Form).
*   Kết hợp các thư viện bên thứ 3 mượt mà, không gây xung đột code.
